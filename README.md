# Discover Coimbatore

## Project Overview
This project is a tourism guide website for Coimbatore, showcasing its top attractions, cultural highlights, and travel planning resources. The site is designed to be visually appealing and user-friendly, with sections for:

- **Hero Section**: Introduction to Coimbatore.
- **About Section**: Quick city overview and highlights.
- **Attractions Section**: Must-visit places with detailed descriptions.
- **Gallery Section**: Photo gallery of scenic spots.
- **Plan Your Trip Section**: Sample itineraries and travel tips.
- **Contact Section**: Form for inquiries and contact details.
- **Credits Section**: Acknowledgment of contributors.

## Features
- Responsive design for desktop and mobile.
- Local and external image references.
- Interactive navigation and contact form.
- Styled using modern CSS techniques.

## How to Run
1. Ensure you have Python installed.
2. Navigate to the project directory in your terminal.
3. Start a local server using:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and visit `http://localhost:8000`.
